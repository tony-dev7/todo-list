Meu primeiro projeto de lista de tarefas desenvolvido com JavaScript.

- Marcar como concluída

## 💻 Tecnologias

- CSS
- JavaScript

## 📚 Objetivo
Projeto criado para praticar lógica de programação e manipulação do DOM.
