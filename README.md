# 📝 To-do List

Aplicação simples de lista de tarefas feita com HTML, CSS e JavaScript.

## 🚀 Funcionalidades

* ✅ Adicionar tarefas
* ✔️ Marcar como concluída
* 🗑️ Remover tarefas com animação
* 💾 Salvamento automático no navegador (LocalStorage)

## 🎨 Interface

* Layout moderno
* Animações suaves
* Design responsivo básico

* JavaScript
